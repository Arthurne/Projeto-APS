# Projeto de Análise e Projeto de Sistemas - UFPB

## Página Inicial

Somos alunos do curso de Sistemas de Informação da Universidade Federal da Paraíba (UFPB).

Fazemos parte de um projeto da disciplina de Análise e Projeto de Sistemas, onde temos o objetivo de analisar, modelar e desenvolver soluções para atender aos requisitos de um cliente real, utilizando práticas e conceitos da Engenharia de Software.

---

# Visão Geral do Projeto

Nosso projeto tem como objetivo desenvolver um sistema capaz de solucionar problemas identificados pelo cliente, aplicando técnicas de levantamento de requisitos, modelagem, prototipação e desenvolvimento de software.
